# MHW-CCE
The files contains the codes used in Chen et al: Multi-trophic level responses to marine heatwave disturbances in the California Current Ecosystem
These codes can mainly be catgorized into three parts, storaged as three branches on this repository: 1) Satellite data processing ,2) _In situ_ data processing and 3) Main analyses. Each branches have their own readme file to explain the propose of each codes and elborate the order of running the codes. The links of raw data source and the required packages/functions are also indicated in the code comments in each codes.   
Please note that some samplings are ongoing projects (e.g. ZooScan), so new raw data from each website may be updated periodically.
